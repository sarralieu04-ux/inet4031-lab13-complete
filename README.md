# INET4031_lab12 - Alieu Sarr

This repository contains the completed Dockerization of a three-tier application: Apache, Flask, and MariaDB, which I completed for my INET4031 Lab 12. The orchestration with Docker Compose functions as a unified ticket system. The lab involved configuring Dockerfiles, creating an .env file, and verifying the stack through manual validation and a check script. For details on the specifics concerning the deployment, validation checking, and design setup, please checkout the full repo!
